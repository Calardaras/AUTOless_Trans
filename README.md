AUTOless_Trans
===================================
自用P社游戏mod手动机翻

用法
-----------------------------------
1.将localization里的english文件夹放入original，运行compose.py<br>
2.手动将生成在process文件夹内的vaule.docx交由第三方翻译器翻译，并更名为tr_va放回<br>
3.运行decompose.py，输出结果在output内<br>

代办集
-----------------------------------
1) <del>遍历子文件夹</del>
2) 自动以1w或5k字分割word文档
3) 适配paratranz的json
